# 商品管理システム

## 概要
このシステムでは、店舗で扱う商品の管理を行うことができます。
商品の新規登録、編集、削除を行うことができ、登録した商品を検索することもできます。
アカウントの権限設定によって使える機能を制限することも可能です。

## 主な機能
- ログイン・ログアウト機能
- 商品一覧機能
- 商品の新規登録、編集、削除機能
- 商品検索機能
- アカウントの権限付与機能

## 開発環境
```
PHP 7.4
MySQL 5.7.39
Laravel 8

```
